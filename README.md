# SpacEscape
## Desarrollo de un videojuego interactivo usando Python, estructurado con múltiples niveles y mapas variados, así como la inclusión de enemigos y desafíos estratégicos. Este videojuego tiene un enfoque educativo y está orientado a potenciar las habilidades cognitivas de los jugadores en áreas como matemáticas, lógica y fundamentos de programación.
